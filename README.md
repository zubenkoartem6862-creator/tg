# Whisper.team Telegram Bot — iPhone Edition

Версия без папок и без `__init__.py`. Все Python-файлы находятся в одном месте, поэтому проект удобно загружать на GitHub с iPhone.

## Что есть

- анкета участника через FSM;
- загрузка скриншотов TikTok и Roblox-скина;
- поддержка с ответами администратора;
- админ-панель, принятие и отклонение заявок;
- блокировки, статистика и рассылка;
- SQLite;
- защита от спама;
- современные кнопки и оформление.

## Переменные Railway

Добавьте в Railway → Variables:

- `BOT_TOKEN` — токен от BotFather;
- `ADMIN_IDS` — Telegram ID администраторов через запятую, например `123456789,987654321`;
- `CHANNEL_URL` — ссылка на канал, например `https://t.me/wh1sp3r_team`.

Не используйте `ADMIN_ID`: правильное название — `ADMIN_IDS`.

## Запуск

Railway запускает:

```bash
python main.py
```

Также можно использовать `python bot.py` — этот файл добавлен для удобства.
